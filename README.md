# Description

This repository contains self-written lecture notes covering various mathematical concepts. The material will be updated regularly.

You can access the content by opening the `main.pdf` file in the corresponding folders.

Most of the material assumes at least a **strong foundation in basic mathematical principles**; therefore, these notes are **not intended to be fully self-contained**.

The project follows two main objectives:

- Present mathematical concepts with a clear and coherent flow.
- Provide detailed proofs for results that are too often left as an “exercise for the reader.”\_
